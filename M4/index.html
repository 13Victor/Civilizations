Civilizations Development Timeline

The road to success was never easy

  • First Step
    Planning and role assignment
    First, we identified the characteristics of our project and divided the work into sections. Then, we assigned a primary role to each worker who would be the main person responsible for their assigned part.
  • Second Step
    Creating the repository on GitHub
    We created a repository on GitHub to store and manage our code. We set up access permissions and organized the project structure.
  • Third Step
    Creating the database
    We designed and created the database needed for the project. We defined the tables, relationships, and initial data required for the system's functionality.
  • Fourth Step
    Development of the game logic
    We developed the game logic, including the main rules, win and loss conditions, and the player's interaction with the system.
  • Fifth Step
    Graphic interface design and creation
    We designed and created the graphical user interface, ensuring it was intuitive and attractive. We included graphical elements, buttons, and menus necessary for interaction.
  • Sixth Step
    Development of the game's website
    We developed the game's website, where users can get information, download the game, and receive updates. We ensured the site was responsive and easy to navigate.
